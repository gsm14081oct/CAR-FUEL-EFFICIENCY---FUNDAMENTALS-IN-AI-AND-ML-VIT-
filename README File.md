# Car Fuel Efficiency Prediction using Machine Learning

This project implements a machine learning regression model to estimate vehicle fuel efficiency (Miles Per Gallon - MPG) based on important vehicle characteristics. The objective is to demonstrate how supervised learning algorithms can be used to solve real-world prediction problems.



The model is built using Python and Scikit-learn and evaluates performance using standard regression metrics. This project is developed as part of a Fundamentals of AI and Machine Learning course.

&#x20;

## Project Overview

This project predicts the fuel efficiency (MPG - Miles Per Gallon) of a car using machine learning techniques.

## 

## Problem Statement

Fuel efficiency is an important factor in vehicle design and environmental sustainability. Predicting fuel efficiency based on vehicle characteristics helps engineers design efficient vehicles.

## 

## Features Used

* Engine Size
* Vehicle Weight
* Horsepower
* Number of Cylinders

## 

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## 

## How to Run the Project

1. Install dependencies:



pip install -r requirements.txt



2. Run the model:



python model.py



## Output

The model prints:

* Mean Squared Error
* R2 Score
* Example MPG Prediction
